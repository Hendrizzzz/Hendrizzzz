## Featured Work

### [Ghostify](https://github.com/Hendrizzzz/Ghostify)

Browser extension for Facebook, Instagram, and Messenger privacy controls. It hides seen receipts, typing indicators, and story views, and is published on both the Chrome Web Store and Microsoft Edge Add-ons.

<p align="center">
  <img src="./large_tile_1400x560.png" alt="Ghostify preview" />
</p>

What I built:

- Published a Manifest V3 extension with 500+ Chrome users and an Edge listing
- Built local-only request and content-script logic with no backend data collection
- Packaged Chromium builds with an ESBuild pipeline and modular service-worker/content-script structure
- Implemented browser-local settings storage for user controls
